# Dotfiles

## usage

```sh
git clone <url-to-this-repo>
cd ~/dotfiles

# stow <package>
stow neovim
```

## requirements

- GNU stow

### neovim

- neovim 0.5.1
- [elixir-ls](https://github.com/elixir-lsp/elixir-ls) (installed at `~/elixir-ls/rel`, see [this reference](neovim/.config/nvim/lua/lsp.lua))
